프롬프트 안의 [제품명], {톤앤매너}, {{배경}} 같은 값을 자동으로 변수로 인식합니다.

GitHub 저장 설정

저장소의 Contents 권한이 있는 fine-grained token을 입력하면, 등록한 프롬프트와 WebP 이미지가 GitHub 저장소에 저장됩니다.

필요 권한
Fine-grained token 기준: Repository access는 creod_promptbox, Contents 권한은 Read and write.
